# LandingPreset

Шаблон приложения для быстрой разработки SSR-лендингов на Next.js. 
Используемые технологии:
- Next.js (SSR) + Typescript
- CSS-modules
- Yarn4 (node_modules, без PNP)

## Генерация секций
Для быстрой генерации секции на сайте необходимо использовать скрипт `yarn section:create <название-секции->`.
Данный скрипт создаст директорию с секцией в папке `sections`, добавит туда базовый серверный компонент секции (.tsx)
и создаст файл стилей в этой же директории. **ВАЖНО**: название секции должно быть с большой буквы, как в стандартах React.
